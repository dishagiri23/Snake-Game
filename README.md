# 🐍 Snake Game

A classic **Snake Game** built using **HTML**, **CSS**, and **JavaScript**. 🎮🚀

## 📌 Features
- 🎮 Smooth gameplay
- 🎨 Simple and clean UI
- 📱 Responsive design
- 🎵 Sound effects
- 🔥 Score tracking

## 🖥️ Tech Stack
- **HTML** 🏗️
- **CSS** 🎨
- **JavaScript** ⚡

## 🚀 How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/dishagiri23/Snake-Game.git
   ```
2. Open the `index.html` file in your browser.
